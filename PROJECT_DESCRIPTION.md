**Project Name:** Share Talk, Heal - Mental Health Support Platform

**Project Type:** Web Application (React + Firebase)

**Tagline:** A safe, anonymous platform for mental health support, meditation, and community connection

**Description:**
Share Talk, Heal is a comprehensive mental health support web application built with React and Firebase. It provides a safe, anonymous space for individuals to share their feelings, connect with peers, and access mental wellness resources. The platform features real-time community interaction, an interactive meditation timer with ambient sounds, and curated self-care resources.

**Key Features:**

1. Anonymous Sharing System
   - Users can post their thoughts and feelings anonymously
   - Real-time community posts powered by Firebase Realtime Database
   - Complete privacy and security guaranteed
   - No login required for basic features

2. Interactive Meditation Timer
   - Three breathing techniques: Box Breathing (4-4-4-4), 4-7-8 Breathing, and Calm Breathing
   - Five ambient nature sounds: Rain, Ocean Waves, Forest, Wind, and Stream
   - Real-time progress tracking with breath counter, phase timer, and progress bar
   - Session history tracking stored in local storage
   - Volume control for ambient sounds
   - Completion celebration modal with statistics
   - Visual breathing guide with animated circle

3. Peer Support Chat
   - Live chat functionality to connect with trained peer mentors
   - Real-time messaging powered by Firebase
   - Anonymous chat option available
   - 24/7 community support

4. Wellness Resources
   - Yoga Practices: Detailed yoga poses with instructions, benefits, and difficulty levels
   - Music Therapy: Curated playlists for relaxation and mental wellbeing
   - Self-Care Tips: Daily practices and advice for mental health
   - Inspirational Quotes: Uplifting words and motivation

5. Community Features
   - Anonymous notes board where users can read supportive messages
   - Real-time updates of community posts
   - Ability to share experiences without judgment

**Technology Stack:**

Frontend:
- React 18.x
- React Router for navigation
- CSS3 with animations and transitions
- Responsive design (mobile-first approach)

Backend:
- Firebase Realtime Database for real-time data synchronization
- Firebase Hosting for deployment

APIs & Services:
- Pixabay Audio API for ambient meditation sounds
- Web Audio API for sound playback and volume control

Storage:
- Local Storage for session history persistence
- Firebase for community posts and chat messages

**Technical Highlights:**

- Professional SVG icons throughout the application
- Glassmorphism and gradient design effects
- Smooth animations and hover effects
- Real-time data synchronization
- Session tracking and progress monitoring
- Fully responsive design for all devices
- WCAG accessibility compliant
- No authentication required for basic features

**Project Structure:**

Main Components:
- Header: Navigation with links to all sections
- Home: Landing page with hero section, features, statistics, and testimonials
- MeditationPage: Full interactive meditation timer with all features
- YogaPage: Yoga poses with detailed instructions
- MusicPage: Music therapy playlists
- SelfCarePage: Self-care tips and practices
- QuotesPage: Inspirational quotes
- AnonymousNotes: Community posts board
- ShareModal: Modal for creating anonymous posts
- ChatModal: Live chat interface
- Resources: Resource cards linking to different pages

**Installation Requirements:**
- Node.js (v14 or higher)
- npm or yarn
- Firebase account (free tier works)

**Setup Steps:**
1. Clone the repository
2. Run npm install
3. Configure Firebase (create config file with Firebase credentials)
4. Run npm start
5. Access at localhost:3000

**Environment Variables Needed:**
- Firebase API Key
- Firebase Auth Domain
- Firebase Database URL
- Firebase Project ID
- Firebase Storage Bucket
- Firebase Messaging Sender ID
- Firebase App ID

**Deployment:**

- Build command: npm run build
